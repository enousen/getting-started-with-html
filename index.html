Unit 1: Basics of the web

World Wide Web
Collection of HTML documents connected by hyperlinks, ~30 billion pages.
Made up of browser requests via internet to servers using HTTP.
Servers respond with files that browser displays
Servers host files

HTML – hypertext markup language
contains text content – what you see - and markup – what it looks like
Tags format: < name >(opening tag) content < /name > (closing tag)
Both tags and content form an element

Examples: bold - < b >, italic < em >

Attributes: < tag ATTR=”value”>contents< /TAG >

Examples: anchor (to make link): < a href="www.google.com">, images < img src= "url" alt = "text">

"url" shows where image can be found, "alt" shows text when picture can’t be displayed (broken link, blind people)

If a tag has no end tag, it is a void tag (no content, so doesn’t need closing tag)

White Space
all spaces (tabs, enters, multiple spaces) are viewed as a single space in HTML
Use < br > (break) tag or < p > (paragraph) tag

The paragraph tag is a block tag which allows you to customize the height, weight of content within the tag, instead of just added an enter like a break tag

Inline tags: < b >, < em >, < br >, < img >, < strong >, < a >
Block tags: < p >
Container tags: < span > (inline); < div > (block)

HTML documents contain: doctype, HTML tag, head tag (CSS styling goes here), title tag (title of page in top of browser window/tab), body tag (content of document)
